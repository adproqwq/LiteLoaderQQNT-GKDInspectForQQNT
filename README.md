# LiteLoaderQQNT-GKDInspectForQQNT

## 说明
使用文件导入的方法无法获得快照分享链接，

快照链接导入不受影响。

支持收集QQ聊天中的快照链接，可以在插件设置页查看。

## 安装方法
1. 下载最新 [发行版](https://github.com/adproqwq/LiteLoaderQQNT-GKDInspectForQQNT/releases) 并解压
2. 将文件夹移动至 `LiteLoaderQQNT数据目录/plugins/` 下面
3. 重启QQNT即可
